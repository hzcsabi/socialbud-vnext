# Static assets

Put images and other static files here. They are served from the site root.

**Examples:**
- `public/logo.svg` → use as `/logo.svg` or `<Image src="/logo.svg" ... />`
- `public/images/logo.png` → `/images/logo.png`

Use the Next.js `<Image>` component from `next/image` for optimized loading (optional).
